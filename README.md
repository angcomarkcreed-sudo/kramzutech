### 💡 About Me

I am a Computer Science student at the University of Caloocan City (UCC) who treats application interfaces like complex, perfectly organized hubs—everything needs to be exactly where the user expects it. I believe functional software only truly succeeds when the interface is highly intuitive. 

- 🔭 **Currently working on:** - **SariSmart:** An inventory and sales management system designed specifically for micro-retailers.
  - **UART:** A dedicated campus lost-and-found and recovery system for UCC.
- 🌱 **Currently obsessed with:** Modernizing layouts, custom-styled UI components (CSS, Windows Forms), and Android design elements (9-patch graphics).
- ⚙️ **Under the hood:** I also have a strong academic interest in processor architecture, instruction sets (ARM, MIPS), and memory organization.
- 💬 **Ask me about:** User-centric design principles, UI modernization, or how to structure the perfect MMO storage vault.
